BASICO

1. Lo primero que se hace es crear el repositorio en GitHub
2. git clone “Enlace del repositorio”
3. git status --- Permite revisar el estado del repositorio, es decir, se pueden ver los archivos que se encuentran en el repositorio.
4. git add . -O- git add "Nombre del archivo" --- Esto permite añadir un archivo o los archivos modificados.
5. git commit -m "Mensaje que explica que se ha hecho" --- Esto permite añadir la version y el mensaje de lo que se realizo, ya queda guardada la version.
6. git push --- Esto permite subir la version que se ha actualizado al gitHub, es decir, a la nube.

BRANCHS

1. git branch "Nombre de la rama" --- Esto permite crear una rama (branch)
2. git checkout "Nombre de la rama" --- Esto permite cambiar de rama.
3. git push origin "Nombre de la rama" --- Esto permite realizar el comando push desde la rama "Nombre de la rama"
